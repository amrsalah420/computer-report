Operating system is a software that manages all the hardware and software processes that happens in a computer and provides common services for
computer programs. The operating system manages software hardware processes such as:
1) Input devices like keyboard and mouse. 2) Output devices like monitors and printers. 3) Storage devices such as internal and external drives.

The OS also provides services to facilitate the efficient execution and management of memory allocations. If several programs are running at
at the same time, the OS will allocate the computer's resources to make sure that each one of them receives what it needs to function.
Some operating systems were developed in the 1950s, when computers could only execute one program at a time. Later in the decade, computers included many
software programs which were linked together to create the beginning of today's operating systems.

Operating system components:


kernel: This provides basic-level control over all of the computer hardware devices. Its main roles include reading data from memory and writing
datat to memory, processing execution orders, determining how data is received and sent by devices, such as the monitor, keyboard, and mouse and determining
how to interpret data recieved from networks.

User interface: This component allows interaction with the user, which may occur through graphical icons and a desktop or through a command line.
The UI is further devided into command line interface, consisting of a text-based interface where advanced users can prompt specific commands by typing them,
and then devides into Graphical user Interface.

Application Programming interfaces: This component allows application developers to write module code.
